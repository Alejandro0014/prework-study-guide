HTML

  • HTML = Hyper Text Markup Language
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The ul element define an unordered list
  • The ol element defines an ordered list
  • The li element defines a list item

CSS

  • CCS = Cascading Style Sheets
  • CSS defines the colors, aesthetcs, visual layout and organization of a webpage
  • inline CSS: to style an element direcly in the HTML file
  • internal CSS: used to define the style inside the head section of an HTML page
  • external CSS: can be used t add styling to multiple HTML pages at a time
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone: Creates a copy of an existing Git repository
  • git add -A -> git commit -> git pull origin main -> git push origin "branch"

JavaScript

  • Most popular programming language
  • JavaScript allows you to make webpages interactive
  • A falsy value: 0, -, On, "", null, undefined, NaN
  • A truthy value: All values that are considred true
  • A variable named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
